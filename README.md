# InfoBot 🤖

InfoBot is an AI-powered assistant that helps developers resolve programming errors quickly.

## 🚀 Features
- Error detection and explanation
- Suggested solutions
- Related video tutorials
- Beginner-friendly guidance

## 🛠️ Tech Stack
- ASP.NET / C#
- HTML, CSS, JavaScript
- AI integration

## 📌 Future Improvements

- Chat-based UI
- API integration
